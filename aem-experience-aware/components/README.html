Component Experience Aware Pattern

Within the components we have added the same property displayExperience.

NextGen components are wrapped with a div that contains a ns-{componentName}-component class name. We have appended this displayExperience as a class name for CSS specificity.

<!-- Component Changes (Image Components) -->
<div class="ns-image-component ${displayExperience}">
    ........

Now in our component's CSS client lib we can target the experience like this:

.ns-image-component.investor .title,
.ns-image-component.investor .title a[href] {
    margin-bottom: 0.71428571428571428571428571428571rem;
    font-family: "AvenirNextLTW01-Regular", Arial;
    font-weight: normal;
    font-size: 26px;
    font-size: 1.857142857142857rem;
    line-height: 30px;
    line-height: 2.142857142857143rem;
}
